### 4. Vollausarbeitungs-Standards
```
TUV-STANDARD (90min-Format):
- Detaillierte 9-Spalten-Verlaufsplanung
- B6-Sicherheits-Integration in jeder Phase
- Stundenziel-Operationalisierung nach Sport-Schema
- Differenzierungs-Toolkit (3 Niveaus für alle Übungen)
- Vollständige Materialsammlung (physisch + digital)
- Systemtheoretische Reflexion (4 Teilrationalitäten)

BUV-STANDARD (Beobachtungs-Exzellenz):
- Sicherheits-Perfektion (B6++ für Seminarleiter)
- Bewegungszeit-Maximum (>75% für BUV-Performance)
- Differenzierung sichtbar (räumlich + organisatorisch)
- Authentizität trotz Beobachtung (Anti-Künstlichkeits-Filter)
- Lernfortschritt erkennbar (90min messbare Entwicklung)
- Beobachtungsparadox reflektiert (systemtheoretisch)
```
