Willkommen bei ArtefaktCraft

Ein Tool zur standardisierten Erstellung pädagogischer Artefakte.

ArtefaktCraft unterstützt Sie bei der Erstellung von Unterrichtsmaterialien, Sequenzplanungen und anderen pädagogischen Dokumenten nach einheitlichen Standards.

Artefakt erstellen Repository-Status
Standardisierte Vorlagen

Verwenden Sie einheitliche Vorlagen für verschiedene Arten von pädagogischen Artefakten, um Konsistenz und Qualität zu gewährleisten.

Dialoggestützte Erstellung

Erstellen Sie Artefakte durch einen geführten Dialog, der Ihnen hilft, alle relevanten Informationen zu erfassen.

Git-Integration

Automatische Versionierung Ihrer Artefakte mit Git für eine transparente Änderungsverfolgung und Zusammenarbeit.

Schnelleinstieg

  1. Wählen Sie einen Artefakt-Typ aus
  2. Füllen Sie die erforderlichen Metadaten aus
  3. Lassen Sie ArtefaktCraft das Artefakt für Sie generieren
  4. Bearbeiten Sie das generierte Artefakt bei Bedarf nach
