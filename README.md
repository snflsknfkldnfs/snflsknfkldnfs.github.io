## 📚 **Verwendung im Unterricht**

### **🎯 Schnell-Einsatz (5 Minuten Setup):**

1. **QR-Codes erstellen:**
   - Öffne: [QR-Generator](qr-generator.html)
   - Klicke: "Alle QR-Codes erstellen"
   - Drucke & laminiere für Stationsschilder

2. **iPad-Setup pro Station:**
   - QR-Code scannen mit iPad-Kamera
   - "Zum Home-Bildschirm hinzufügen"
   - Station läuft offline nach erstem Laden

3. **Unterrichtsdurchführung:**
   - 6 Stationen à 3 SuS mit automatischem Timer
   - 3-Level-Differenzierung je nach Leistung
   - Peer-Learning durch Partnerarbeit

### **📱 Integration in bestehende Systeme:**

#### **Miro-Board Integration:**
```
1. URL kopieren: https://paulad.github.io/snflsknfkldnfs.github.io/unterricht/Sport/...
2. Miro: "iFrame einbetten" → URL einfügen
3. Größe: 800x600px für optimale Darstellung
```

#### **Teams/Classroom:**
- Links direkt in Chat/Aufgaben einfügen
- Funktioniert in allen gängigen Browsern
- Keine App-Installation erforderlich
